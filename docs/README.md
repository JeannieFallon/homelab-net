# Documentation

This directory contains documentation for the homelab project. It is organized into the following sections:

-   `./knowledge-base`: This directory contains general knowledge, guides, and cheatsheets for the tools and technologies used in this homelab.

-   `./post-mortem`: This directory contains post-mortem reports for specific issues encountered during the setup and maintenance of the homelab. Each report details the problem, the resolution, and lessons learned.
